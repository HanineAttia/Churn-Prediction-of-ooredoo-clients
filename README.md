# Customer Churn Prediction
This project predicts customer churn for a telecom company using a 
simulated dataset of 500K clients. Built during a professional internship.

## Tech Stack
- Python, Scikit-learn, Random Forest, XGBoost, MLP (Neural Network)
- Streamlit, Power BI

## How to run
1. Open the notebook in Google Colab
2. Run all cells
